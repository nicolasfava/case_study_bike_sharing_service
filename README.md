Analisi riguardante il comportamento degli utenti di un servizio di bike sharing a Chicago nell'anno 2025. L'analisi si concentra sulla differenza di comportamento fra utenti abbonati al servizio e utenti non abbonati. Il dataset è l'unione di più dataset ognuno riguardante un mese dell'anno. Essendo un dataset molto grande, con più di 5 milioni di righe, per efettuare la mia analisi ho adoperato Python.

L'analisi prevede 3 macro distinzioni : 
- analisi generiche;
- analisi temporale;
- analisi spaziale.
